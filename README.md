# Retail Business Analytics using Python, PostgreSQL & Power BI

## Project Overview

This project presents an end-to-end Retail Business Analytics solution using Python, PostgreSQL, SQL, and Power BI. The objective is to analyze retail customer shopping data, identify purchasing patterns, and generate business insights through data cleaning, SQL analysis, and interactive dashboards.

---

## Objectives

• Clean and preprocess retail customer data.

• Perform exploratory data analysis (EDA).

• Store cleaned data in PostgreSQL.

• Analyze customer behavior using SQL.

• Build an interactive Power BI dashboard.

• Generate actionable business insights.

---

## Tech Stack

• Python

• Pandas

• PostgreSQL

• SQL

• Power BI

• Jupyter Notebook

---

## Project Workflow

• Imported the retail customer dataset into Jupyter Notebook.

• Cleaned and preprocessed the dataset by handling missing values, removing duplicates, and correcting data types.

• Performed feature engineering by creating Age Group and Purchase Frequency columns.

• Conducted Exploratory Data Analysis (EDA) to identify customer purchasing patterns and trends.

• Imported the processed dataset into PostgreSQL.

• Wrote SQL queries to analyze revenue, customer demographics, payment methods, and product categories.

• Designed an interactive Power BI dashboard with KPIs, charts, slicers, and filters.
